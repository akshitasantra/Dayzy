# Dayzy
A SwiftUI-based iOS app to help users track, manage, and visualize their daily activities. DITL allows you to start, edit, and end activities, see a timeline of your day, and view summarized statistics of your time usage.

## Features
- **Today View**
  - Track your current activity in real time.
  - Start activities manually or with predefined “quick start” options.
  - End activities and log duration automatically.
  - View a live timeline of all activities for the day.

- **Stats Dashboard**
  - See a summarized view of your total time spent today.
  - Track your **most time-consuming activities**.
  - Interactive UI with dynamic cards and rows.

- **Video Integration**
  - Placeholder for recording clips associated with activities (expandable for future updates).

- **Activity Management**
  - Add, edit, or delete activities.
  - Set custom start and end times.
  - Automatically calculate activity duration.

- **Themes**
  - Light and dark mode support.
  - Dynamic color styling across all cards, text, and buttons.

- **Database Integration**
  - Uses **SQLite** via `DatabaseManager` for persistent activity tracking.
  - CRUD operations: Create, Read, Update, Delete activities.
  - Fetches daily summaries and statistics efficiently.
